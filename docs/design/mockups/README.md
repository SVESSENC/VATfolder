# Mockups — Basic VAT Registration Flow

This folder contains a very small, early-stage mockup for the VAT registration flow. It's intended as a simple starting point for discussion and handoff to engineering.

- File: basic-mockup.html — clickable wireframe-like HTML showing primary steps: Assessment, Identity, Draft Application, Validate, Submit.

How to use
- Open [basic-mockup.html](basic-mockup.html) in a browser to review the static mockup.
- Use this as the base for higher-fidelity screens or prototype iterations.

Notes
- The mockup is intentionally minimal and styled to look like a wireframe; it is not interactive beyond basic anchors.
- If you'd like Figma-ready screens or annotated image exports, I can produce those next.
