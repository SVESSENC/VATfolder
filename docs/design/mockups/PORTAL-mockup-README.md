# Portal Mockup

This single-page HTML mockup demonstrates the core screens from the portal plan:

- Dashboard: lists applications, quick search, create-new action.
- Create application wizard: 7-step flow matching the plan (Assessment, Identity & Organisation, Business details, Documents, Validation, Review, Submit).
- Operator toggle: stub to show where operator/admin features would appear.

How to review
- Open `portal-mockup.html` in a browser.
- Click "Create new application" to launch the wizard and step through the 7 steps.
